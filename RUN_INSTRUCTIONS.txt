Gym Tracker App — Run Instructions

This project is a Flutter app (web-first) that also runs on iOS/Android later.

Prerequisites
- Install Flutter SDK (includes Dart): https://flutter.dev/docs/get-started/install
- Verify your setup:
  flutter doctor

Quick Start (Web)
  cd /workspace
  flutter pub get
  flutter config --enable-web
  flutter run -d chrome

Run on Mobile
- Android: flutter run -d android (requires Android Studio + SDK)
- iOS: flutter run -d ios (requires macOS + Xcode)

Build for Production
  flutter build web --release      # Web bundle in build/web/
  flutter build apk --release      # Android APK
  flutter build ios --release      # iOS (requires macOS)

Project Structure
  lib/
  ├── main.dart                    # App entry point
  ├── state/
  │   └── app_state.dart           # All state + data models (~2500 lines)
  └── ui/
      ├── root_shell.dart          # Tab navigation + Dynamic Island overlay
      ├── theme/
      │   └── app_theme.dart       # Centralized colors, sizes, durations
      ├── widgets/
      │   ├── app_card.dart        # Shared card containers
      │   ├── animated_widgets.dart # Animation utilities
      │   ├── dynamic_island.dart  # Dynamic Island UI
      │   └── month_calendar.dart  # Calendar widget
      └── screens/
          ├── workout_screen.dart   # Active workout logging
          ├── log_screen.dart       # Workout history
          ├── calendar_screen.dart  # Calendar view
          ├── schedule_screen.dart  # Routine scheduling
          ├── progress_screen.dart  # PRs and weekly stats
          ├── goals_screen.dart     # Goals & achievements
          ├── settings_screen.dart  # App settings
          ├── muscle_heat_map_screen.dart  # Experimental heat map
          └── map_routes_screen.dart       # Experimental map routes

Data Storage
- NO login required.
- Data stored locally via SharedPreferences.
  - Mobile: platform key/value storage
  - Web: browser localStorage
- Data persisted as JSON in a single 'gym_tracker_db_v1' key.

Key Features
- Dynamic Island UI with gesture controls
- Workout tracking with smart rest timers
- Progressive overload detection
- Achievement system with 12 unlockable badges
- Fitness goals with milestones
- Muscle heat map (experimental)
- Animated UI with confetti, pulsing glows, staggered lists

Development Tips
- Run lints: flutter analyze
- Run tests: flutter test
- Hot reload: Press 'r' in terminal or save files in IDE

Troubleshooting
- "flutter not found": Add Flutter SDK bin/ to your PATH
- "No devices found": flutter config --enable-web && flutter devices
- "Pub get failed": Delete pubspec.lock and re-run flutter pub get

Notes for Remote Environments
- This workspace may not have Flutter installed.
- Clone the repo locally and run with Flutter on your machine.
- All code is pure Dart/Flutter with no native dependencies (except flutter_map for experimental features).
